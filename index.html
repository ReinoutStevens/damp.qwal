qrpe

macro
Usage: (qrpe graph start end bindings & exps)

A macro on top of solve-qrpe that allows for nicer syntax.
Graph holds the graph, and should at least understand :nodes
Start and end are unified with nodes in graph.
Bindings are the new introduced variables that are kept throughout the pathexpression.
Exps are the actual goals that should hold on the path through the graph.
Each goal should be a rule that takes 2 variables.
First variable is the current world, and will be ground.
Second variable is the next world, and goal must ground this.

solve-goal

function
Usage: (solve-goal goal curr next)

calls goal with curr and next. curr is a grounded node. goal should ground next

solve-goals

function
Usage: (solve-goals goals curr end)

goals is a list of goals. Each goal is a rule with two arguments, current and next world.
Each goal is called, passing the next version of the previous goal as the
current version of the current goal

solve-qrpe

function
Usage: (solve-qrpe start end & goals)

main rule that solves a qrpe

to-node

function
Usage: (to-node node to)

succeeds when to is the list of nodes that are direct successors of node

trans

function
Usage: (trans node next)

succeeds when next is a direct successor of node

with-current

macro
Usage: (with-current [current] & goals)

macro that evaluates a series of goals in the current world. current is bound to the current world
